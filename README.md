# SwarmUI-rtx-upscale
This is a poorly vibecoded extension to add support for the rtx upscale node in SwarmUI. Next to 0 QA.

This requires my custom rtx node to be installed into comfyui first. It doesn't do this on its own because I didn't feel like putting in more effort. I didn't use the official node because enums are hard.
